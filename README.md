# elo-data


ローグライクゲームElonaのヴァリアントにおける変数使用状況の調査 (リポジトリ作成現在は本家との差分のみ)

その性質上スポイラー、ネタバレを多量に含みます



※データの公開方法は将来的に変更する可能性があります
