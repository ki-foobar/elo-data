URW
    eqsc.s1
        i32 x (5 x 16) x 10
        5: 1アイテムあたりの情報
        16: 1装備セットあたりの最大アイテム数
            おそらくカオスシェイプ PC の最大装備枠数を基準とした数値
        10: 装備セット最大数

        0 アイテムのID + 1
          0 なら装備未設定
        1 部位ID
        2 アイテムの品質
        3 アイテムの素材
        4 アイテムの銘/エゴ
          品質が 3 (高品質) 以上の場合にのみ使用される


    urw_inv.s1
        i32 x 2 x 5480

        0 武器に塗った媚薬の残り回数
          塗ったときに 5 + rnd(5) 足され、攻撃のたびに減っていく。0 になると切れる
        1 名前の記された友情の証における対応しているキャラ番号
          0 の場合は本家と同じ無名の友情の証 (URWで生成されるものには必ず名前がつくが、本家などから移行した場合はその限りでない)


    urw_memo.ud
        テキストデータ

        URW の終了時にプレイメモを残す機能のための変数。次回起動時に表示される


    urw_gdata1.ud
        i32

        今月購入できるプラチナ硬貨の残り枚数。この機能については URW の readme に詳しい
